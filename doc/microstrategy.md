# MicroStrategy connector

## Connector configuration

* `type`: `"MicroStrategy"`
* `name`: str, required
* `base_url`: str, required
* `username`: str, required
* `password`: str, required
* `project_id`: str, required


## Data source configuration

* `domain`: str, required
* `name`: str, required
* `id`: str, required
* `dataset`: Dataset, required
