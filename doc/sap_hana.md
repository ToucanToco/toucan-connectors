# Sap Hana connector

Import data from Sap Hana.

## Connector configuration

* `type`: `"SapHana"`
* `name`: str, required
* `host`: str, required
* `port`: str, required
* `user`: str, required
* `password`: str, required


## Data source configuration

* `domain`: str, required
* `name`: str, required
* `query`: str (not empty), required
